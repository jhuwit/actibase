# Vector of X, Y, Z, and maybe time

Vector of X, Y, Z, and maybe time

## Usage

``` r
xyz

xyzt

txyz
```

## Format

A vector with 3 or 4 elements, which are:

- X:

  value to extract X column

- Y:

  value to extract Y column

- Z:

  value to extract Y column

- time:

  value to extract time column

An object of class `character` of length 4.
