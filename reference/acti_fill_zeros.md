# Fill in Zeros

Fill in Zeros

## Usage

``` r
acti_fill_zeros(data)

acti_fill_zeroes(data)
```

## Arguments

- data:

  a data frame containing columns time, X, Y, Z

## Value

the modified data frame with zeros replaced by NA
